Manager Dashboard

This is a Template Page for a Manager Microservice

The Manager consists of different Components, each running as own Microservices

Here you see various approaches of "importing" or "including" the Snippets.

1. With SSI and nginx as Reverse Proxy

2. With HTML Imports (from Web Components) and nginx as Reverse Proxy

"Experimental Web Platform feature" Flag in Google Chrome must be enabled. (Goto chrome://flags/)

3. With HTML Imports, Polymer and nginx as Reverse Proxy

See https://github.com/webcomponents/webcomponentsjs#browser-support
